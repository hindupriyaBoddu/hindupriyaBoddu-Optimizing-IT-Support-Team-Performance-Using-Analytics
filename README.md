🎓 Student Performance Analysis Dashboard

 📌𝐏𝐫𝐨𝐣𝐞𝐜𝐭 𝐎𝐯𝐞𝐫𝐯𝐢𝐞𝐰:
The **Student Performance Analysis Dashboard** is a data analytics project developed during the **Infosys Springboard Data Analytics Internship**.  
The objective of this project is to analyze student academic performance using data analytics techniques and visualize meaningful insights through an interactive **Power BI dashboard**.
This project demonstrates how raw educational data can be transformed into actionable insights that help educators, institutions, and decision-makers understand factors influencing student performance.

🎯𝐏𝐫𝐨𝐣𝐞𝐜𝐭 𝐎𝐛𝐣𝐞𝐜𝐭𝐢𝐯𝐞𝐬:
✅ Analyze student performance across subjects  
✅ Identify factors affecting academic scores  
✅ Perform data cleaning and transformation using Power Query  
✅ Apply Exploratory Data Analysis (EDA) concepts  
✅ Build interactive dashboards using Power BI  
✅ Present insights using data storytelling techniques

 Column Name  ------> Description

 Test Preparation ------>Completion status of test preparation course ( Completed or not)
 Math Score   ------>Mathematics exam score 
 Reading Score------> Reading exam score
 Writing Score ------>Writing exam score
